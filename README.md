# SavsBakedGoods
Project Zomboid Mod to add a few Breads

Adds Craftable Versions of Vanilla Items
  - Basic `Sandwich` Bread
  - Cornbread
 
Adds new bread items!
  - Sourdough
  - Soft Pretzels
  - Hard Tack
  - Fluffy Milk Bread
  - Matzo
  - Matzo Balls (for Soup)
  - Egg Salad (To add to sandwiches)
  - HorseshoeSandwich
  
Adds new bread tools!
  - Lame
